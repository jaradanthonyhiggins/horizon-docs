## What Is The .session Command?

The `.session` command lets you **search for session edit(s)** and sends a detailed embed showing **song title, era, cover art, and the audio file itself**.  
If multiple songs match the search term, a **dropdown menu** will let you choose the correct song.

---

## Getting Started

<Steps>
    <Step title="Ensure Permissions">
        <Info>
            Make sure **Horizon, the channel, or the role** has the permissions **"Send Messages"** and **"View Channel"** in the channel where you want to run the command.
        </Info>
    </Step>
</Steps>

---

## How It Works

<Info>
Input any **song name** to fetch matching session edits.  

---

Horizon will search for song title(s) with a surfaced session edit.  
If multiple matches are found, a **dropdown menu** will allow you to select the correct session.
</Info>

<CodeGroup>
```javascript Syntax
.session (songname)
```

```javascript Example
.session Hypnotic
```
</CodeGroup>

---

## Handling No Matches

<Warning title="No Sessions Found">
If no sessions are found for the name you entered, Horizon will respond with:  
`No sessions matching '<session name>' found in Sessions!`
</Warning>

---

## Displayed Information

When a session is found, Horizon will display a detailed embed including:

- **Song Title**  
- **Era:** (with matching color & cover art thumbnail)  
- **Preview / Audio File:** Sends the full session file if under **100MB**  

<Info title="Example Embed">

> ### **Song Name:**  
- **Hypnotic (Sessions)**
---

> ### **Era:**
- **Goodbye & Good Riddance**  
</Info>

---

## Navigation & Selection

<Info>
Dropdown selection is limited to **60 seconds** of interaction. If inactive for more than 60 seconds, the menu will disable.
</Info>

---

## Tips & Best Practices
<Tip>Always type **full or partial song names** to get the best match.</Tip><Warning>Sessions larger than 100MB cannot be uploaded.</Warning><Warning>Only the original requester can select from the dropdown menu.</Warning>

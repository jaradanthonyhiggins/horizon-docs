## What Is The Cover Command?

The cover command can send Juice WRLD song and album covers by many very talented artists (Yes, this includes unreleased, unsurfaced, and released songs). 

---

## Getting Started

<Steps>
    <Step title="Ensure Permissions">
        <Info>
            Make sure **Horizon, the channel, or the role** has the permissions **"Send Messages"** and **"View Channel"** in the channel where you want to run the command.
        </Info>
    </Step>
</Steps>

---

## How It Works
<Info>Input The Song Title(s) You Want Covers For, Then Select The Correct Song Title From The Dropdown Below (If Multiple Matches Are Found).</Info>
<Tip>Follow These Simple Steps Above To Learn It In Seconds.</Tip>
<CodeGroup>
```javascript Syntax
.cover (songname)
```

```javascript Example
.cover Broke Boys
```
</CodeGroup>

---

## Handling Multiple Matches

<Info title="Multiple Covers Found">
When multiple covers match your search, the bot displays a dropdown menu with up to **25 options**.  
Select the correct song title to send all associated images.
</Info>

<CodeGroup>
```javascript Syntax
# Multiple matches trigger a dropdown menu for selection
```

```javascript Example
Horizon: Multiple Matching Covers Found:
[Select a cover name]
> Reach
> Reach Freestyle
```
</CodeGroup>

## File Informations & Limits

- Maximum **100 MB per batch** of images.  
- Images are sent in chunks of **10 per message**.  
- Supported formats: ```
.png
.jpg
.jpeg
.gif
.bmp
.tiff
.webp
.ico
.jfif
.avif```

<Warning title="File Size Exceeded">
Please Note That Files Over 100MB Will **NOT** be able to be uploaded to discord.
</Warning>

---

## Tips & Best Practices

- Use **Official Song Titles** for better results.  
- **Partial names work:** pick the correct one from the dropdown.  
- **Works** with **all major** image types: `.png`, `.jpg`, `.jpeg`, `.gif`, `.bmp`.

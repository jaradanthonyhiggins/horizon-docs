## What Is The Unsurfaced Command?

The unsurfaced command lets you **view every currently unsurfaced / unleaked songs in era and alphabetical order and cycle throug them all with navigation buttons**.  

---

## Getting Started

<Steps>
    <Step title="Ensure Permissions">
        <Info>
            Make sure **Horizon, the channel, or the role** has the permissions **"Send Messages"** and **"View Channel"** in the channel where you want to run the command.
        </Info>
    </Step>
</Steps>

---

## How It Works

<Info>
Once .unsurfaced is used, it will return an embed with buttons to navigate to the next unsurfaced song. It is in era order and alphabetical order.  
</Info>

<CodeGroup>
```javascript Syntax
.unsurfaced
```

```javascript Example
.unsurfaced
```
</CodeGroup>

## Navigation & Pagination

When multiple tracks match, Horizon shows **1 track per page** with navigation buttons:

- ⬅️ **Previous Page**  
- ➡️ **Next Page**  

<Info>
Pagination is limited to **60 seconds** of interaction. If inactive for more than **60 seconds**, the buttons will be disabled.
</Info>

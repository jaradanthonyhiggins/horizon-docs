## Viewing information about a cosmetic

You can use the `fortniteitem` command to view information about a cosmetic in Fortnite.

<CodeGroup>

```javascript Syntax
.fortniteitem (cosmetic)
```


```javascript Example
.fortniteitem Festival Lace
```

</CodeGroup>

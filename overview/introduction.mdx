# Introduction

<Banner variant="info">
Horizon is a multi-purpose Discord bot built to simplify server management and enhance community engagement.  
This documentation will guide you through installation, configuration, and advanced usage.
</Banner>

---

## Quick Start

<Steps>
1. **Invite Horizon** – Add the bot to your server using the official invite link.
2. **Check Prefix** – The default prefix is `.`. You can change it with `,prefix set !` or another symbol.
3. **Run Setup** – Use `.setup` to begin configuring Horizon, or visit the dashboard if enabled.
</Steps>

<Warning>
The default prefix is `.`. If you already use another bot with the same prefix, change it immediately to avoid conflicts.
</Warning>

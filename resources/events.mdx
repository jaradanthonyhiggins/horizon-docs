---
title: Events
description: "All available toggleable events within bleed."
icon: location-dot
---

| Event                 | Description                                            |
| --------------------- | ------------------------------------------------------ |
| `y/n`                 | Toggle upvote/downvote to messages.                    |
| `v/s`                 | Toggle left arrow/right arrow to messages.             |
| `afk`                 | Toggle away messages when pinging AFK members.         |
| `previousreaction`    | Toggle any previous reaction triggers.                 |
| `reactiontrigger`     | Toggle any reaction triggers.                          |
| `autoresponder`       | Toggle all auto responders.                            |
| `commandfailure`      | Toggle failed command execution messages.              |
| `snipe`               | Toggle messages from being sniped completely.          |
| `cleverbot`           | Toggle clever bot conversations.                       |
| `automodmessage`      | Toggle bleed auto-mod messages upon user punishment.   |
| `instagram`           | Toggle Instagram reposting from members.               |
| `tiktok`              | Toggle TikTok reposting from members.                  |
| `grailed`             | Toggle Grailed reposting from members.                 |
| `twitch`              | Toggle Twitch reposting from members.                  |
| `streamable`          | Toggle Streamable reposting from members.              |
| `twitter`             | Toggle Twitter/X reposting from members.               |
| `medal`               | Toggle Medal reposting from members.                   |
| `soundcloud`          | Toggle SoundCloud reposting from members.              |
| `tumblr`              | Toggle Tumblr reposting from members.                  |
| `shorts`              | Toggle Shorts reposting from members.                  |
| `kick`                | Toggle Kick reposting from members.                    |
| `youtube`             | Toggle YouTube reposting from members.                 |
| `reddit`              | Toggle Reddit reposting from members.                  |
